# Quick: Manually Deploy in Vercel

## Fast Solution

1. **Go to:** https://vercel.com/dashboard
2. **Select:** Your project (`cma-leader-dashboard`)
3. **Click:** "Deployments" tab (top navigation)
4. **Click:** "Deploy" button (top right, blue/green button)
5. **Select:** Branch `main` (if asked)
6. **Click:** "Deploy"
7. **Wait:** Watch the deployment start

That's it! This will use the latest commit from GitHub (with the fixed package.json).

---

**No need to wait for auto-deploy - just click "Deploy" manually!**
